Parametros del informe

La generacion consulta tickets del periodo seleccionado y envia el contexto a Gemini desde la API.

Notas operativas

El endpoint esta protegido para administradores desde la API y desde la ruta Angular.

Si Gemini no tiene API key configurada, la API devolvera error al intentar generar el informe.

El limite de tickets controla cuanta muestra detallada se envia al modelo. Usa valores altos solo cuando el periodo lo justifique.
